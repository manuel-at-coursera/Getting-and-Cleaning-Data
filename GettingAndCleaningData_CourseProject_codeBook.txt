Column 1: Subject
* This column contains numeric values from 1 to 30.
* Assignes the number of the participant (subject) to whom the row of data belongs.

Column 2: Activity
* Per row, this column contains one of the following character values:
  * walking
  * walking_upstairs
  * walking_downstairs
  * sitting
  * standing
  * laying
* Assignes the activity to which the row of data belongs.
  
The remaining columns (3 to 68) contain numeric values with the following characteristics:
* They are obtained from a accelerometer ("Acc") or a gyroscope ("Gyro").
* They contain the X-, Y- or Z-axis component ("-X", "-Y", "-Z") or the magnitude of the corresponding value ("Mag").
* They contain the mean ("mean()") value or the standard deviation ("std()").
* The value refers to the body movement of the subject ("Body") or to the gravity ("Gravity").
* "Jerk" assignes jerk signals.
* Some column labels start with "f" (instead of "t"), assigning that those values were obtained by a Fast Fourier Transform.
